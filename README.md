# Teslo API

1. Clonar Proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Ajustar las variables de entorno
5. Levantar la Base de Datos
```
docker-compose up -d 
```
6. Ejecutar SEED
```
http://localhost:3000/api/seed
```
7. Levantar: ```yarn start:dev```
